# 文案模板

## 一、标题公式（每次从不同类型各出 1 条，共 3 条备选）

| 类型 | 模板 | 日语学习示例 |
|---|---|---|
| **数字型** | N个XX，第M个我到日本才知道 | 8个理发店日语，第5个中国人全说错 |
| **反认知型** | XX千万别说成YY / 原来XX是错的 | 「大丈夫です」千万别乱用，日本人会误会 |
| **场景型** | 在日本XX时，日本人其实这么说 | 在日本便利店结账时，店员到底在问什么 |
| **圈人群型** | 去日本旅游/N级党/留学生 必看 | 去日本旅游只背这9句就够了 |
| **不看会亏型** | 学了N年日语才发现XX | 学了3年日语才发现，すみません有4种用法 |
| **提问型** | 你知道XX该怎么说吗 | 剪头发时"打薄"日语怎么说？90%的人不会 |
| **对比型** | XX vs YY，差别在哪 | わかりました和承知しました，用错很失礼 |

**标题硬规则**
- 18–22 字最佳，手机上不折行
- 前 8 个字必须有信息（可能被截断）
- 带系列名 + 期数：`｜在日本这样说 第7期`
- 数字用阿拉伯数字（8 比 八 更抓眼）
- 不要用感叹号堆砌，不要"绝了""救命"这类通用感叹（同质化严重）

## 二、封面公式

```
┌─────────────────────┐
│ 在日本这样说 · 第7期   │ ← 左上角，小字，固定位置（系列辨识度）
│                      │
│    在日本理发         │ ← 主标题，超大字 60pt+，≤10字
│    这样说就对了       │
│                      │
│  ▸ 打薄 / 修一修 / 分头 │ ← 副行：3个具体词，制造"点进去能学到"
│                      │
└─────────────────────┘
```

- **配色固定**（和卡片一致）：浅青渐变底 + 深青主标题 + 琥珀点缀
- 一次做好模板，之后每条只换字，5 分钟出封面
- 主标题必须是**大白话**，不要日文（大部分刷到的人不认识日文）

## 三、正文模板

```
在日本理发店，我第一次去的时候完全说不出"打薄"两个字 😅
后来发现日本人根本不用我背的那个词。

今天这 8 句，去一次日本理发店就够用了 ——

1️⃣ 少し短くしてください（sukoshi mijikaku shite kudasai）
   麻烦稍微剪短一点

2️⃣ すきばさみでお願いします（sukibasami de onegai shimasu）
   我想打薄

3️⃣ ……（把所有词全列出来）

⸻

📌 建议收藏，去日本前翻一遍
你还想看哪个场景？评论区告诉我，下期就拍 👇

#日语学习 #日语 #日语口语 #零基础日语 #日本旅游 #日语单词 #日语日常会话 #学日语 #日本生活
```

**正文硬规则**
- **必须把所有词/句在正文里全部列出来** —— 这是收藏的直接原因，也是搜索被收录的关键
- 每句给「日文 + 罗马音 + 中文」三行
- 开头 2 行讲一个具体的个人经历（不要"今天教大家"）
- 结尾抛**具体**问题，不要"欢迎评论"
- 长度 300–600 字

## 四、话题标签配比（6–10 个）

| 层级 | 数量 | 作用 | 例 |
|---|---|---|---|
| 大词 | 2 | 进大池 | #日语学习 #日语 |
| 中词 | 2–3 | 精准人群 | #日语口语 #零基础日语 #学日语 |
| 长尾/搜索词 | 2–3 | **搜索流量入口，最重要** | #日本旅游日语 #日语日常会话 #日语单词 |
| 场景词 | 1–2 | 蹭泛流量 | #日本生活 #日本旅游 |

长尾词要对应真实搜索意图：用户会搜「日本理发怎么说」而不是「日语教学」。

## 五、评论区置顶话术（发布后 30 分钟内自己发）

**纯干货视频（①类）：**
> 完整的 8 句我放正文了，直接复制走 ✌️ 想看哪个场景评论区点单～

**引导关注（每条都可以用）：**
> 这是「在日本这样说」第 7 期，前面 6 期在主页，怕迷路可以先关注 🐟

**转化视频（③类，备案通过后）：**
> 这些内容我整理进了一个日语跟读网站（免费的），**关注后私信「日语」** 发你邀请码 ✌️

> ⚠️ 一定是「**关注后**私信」，这一步把播放量变成粉丝。别只写"私信我"。

## 六、发布时间

| 时段 | 说明 |
|---|---|
| **12:00–13:00** | 午休，学习类内容首选 |
| **19:00–22:00** | 晚间流量高峰，竞争也大 |
| 周末 10:00–11:00 | 学习类周末表现不错 |

固定一个时间日更，让算法和粉丝都形成预期。
