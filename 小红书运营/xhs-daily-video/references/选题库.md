## 系列一『在日本这样说』（场景对话 · 主力）

高潜力场景（按爆款概率排）：

- [ ] 在日本**理发**：想剪短一点 / 不要剪太多 / 打薄
- [ ] 在日本**便利店**：要不要加热、要袋子吗、集点卡
- [ ] 在日本**药妆店**：找某个药、免税、能不能试用
- [ ] 在日本**餐厅**：几位、禁烟席、单点还是套餐、结账（网站已有「餐厅预约」模块）
- [ ] 在日本**问路**：迷路了、换乘、这班车去不去XX
- [ ] 在日本**看病**：哪里不舒服、过敏、几天了
- [ ] 在日本**租房看房**：押金礼金、几站路、能不能养宠物
- [ ] 在日本**打工面试**：能上几天、什么时候能开始
- [ ] 在日本**快递/取件**：不在家再配达、放门口
- [ ] 在日本**酒店**：几点退房、能不能寄存行李、加一床被子
- [ ] 在日本**居酒屋**：先来杯生啤、再来一份、分开结账
- [ ] 在日本**买衣服**：能试穿吗、有没有大一号、这个多少钱
- [ ] **拒绝**的 5 种说法：日本人从不直接说"不"
- [ ] **道歉**的层级：すみません / ごめん / 申し訳ありません 什么时候用哪个
- [ ] 邻居/同事**寒暄**：搬家打招呼、电梯里遇到

反认知型（最容易爆）：

- [ ] 「大丈夫です」到底是 yes 还是 no
- [ ] 点餐说「これください」为什么日本人会愣一下
- [ ] 「頑張って」对正在崩溃的人说是错的
- [ ] 日本人说「考えておきます」= 拒绝
- [ ] 「ちょっと…」后面为什么不说完
