# 小红书涨粉 + 网站变现总策略

> 2026-07-18 制定。目标：粉丝 300 → 1000+，跑通「内容 → 关注 → 试用 → 付费」闭环。

---

## 一、先说三个必须面对的现实问题

在讨论"怎么做爆款"之前，有三件事不解决，粉丝涨上来也转化不了。

### 问题 1：大陆用户打不开你的网站（最致命）

`japanese-learning.edgeone.dev` 是「Global (MLC excluded)」区域，大陆访问返回 401。
**小红书 95% 的流量是大陆手机用户。** 现在的链路是：

> 看到视频 → 有兴趣 → 关注要码 → 拿到码 → **打不开** → 流失 + 可能被骂

ICP 备案卡在第 4 步（云厂商初审），乐观还要 2-3 周。

**在备案下来之前，不要大规模推"来我网站"。**
这段时间的正确做法：**只推内容，攒粉丝，不推产品**。视频里完全不提网站，专心做纯干货，把粉丝池先蓄起来。等域名通了再统一开闸转化。

（临时兜底：Cloudflare `.pages.dev` 那条链路部分网络能通，可以在私信里给"备用链接"，但不要当主推。）

### 问题 2：你的网站要用户自己填 AI Key

翻译/语法解析都依赖用户去 DeepSeek 或智谱申请 API Key 填进设置。
**对小红书上的普通日语学习者，这一步会劝退 90% 的人。**

三个解法，按推荐度排：

| 方案 | 做法 | 成本 |
|---|---|---|
| **A（推荐）** | 内置一个你自己的 Key 做"体验额度"，用完了才引导用户填自己的 | 智谱 GLM-4-Flash 免费，几乎零成本；但 Key 会暴露在前端 |
| B | 做个极简中转（EdgeOne Pages Functions），Key 藏服务端 + 限流 | 要写一点后端，最稳 |
| C | 保持现状，录一条 60 秒「怎么申请免费 Key」教程放置顶 | 零成本，但仍会掉一批人 |

**先做 C 保底，备案期间抽空做 B。**

### 问题 3：备案风险与推广规模是冲突的

CLAUDE.md 里已经记了这条：备案走的是"个人自用、申报从简"的路子。如果小红书真推起来、访问量上去，被举报/抽查的概率会上升。

**建议：现在就把音频挪到腾讯云 COS 默认域名**（CLAUDE.md 里的方案 2）。改动量不大，备案域名上只剩文字和图片，风险敞口小很多。等你有 1000 粉再改就来不及了。

## 二、变现模式建议（你问的那个问题）

先给结论：**分三段走，现在处在第一段，第一段不要收钱。**

### 阶段一：0 → 1000 粉（现在～约 2 个月）目标是"证明有人要"

- 网站**完全免费**，邀请码随便发
- 唯一要求：**关注 + 私信"日语"发码**（这一步是为了把公域流量转成关注和私信关系）
- 真正的目的不是赚钱，是拿到三样东西：**种子用户、真实反馈、可截图的口碑**
- 同步做的事：把拿到码的人**拉进一个微信群**（这是你未来最值钱的资产）

> 为什么不能一开始就收费？300 粉的账号没有信任基础，收费只会让转化率归零，而且你还没有任何用户证言可以放进视频里。

### 阶段二：1000 → 5000 粉，开始收费

推荐 **「免费版 + Pro 版」** 而不是整站收费：

| | 免费版 | Pro 版 |
|---|---|---|
| 跟读练习 | 前 10 篇 | 全部 50 篇 + 英语 10 篇 |
| 日常用语 | 2 个模块 | 全部 16 个场景 / 900 句 |
| 口语练习 | ✅ | ✅ + AI 追问 |
| 语法解析 | 每天 5 次 | 不限 |
| 生词本导出 | ❌ | ✅ |

**定价建议：一次性买断 ¥39–59，或年费 ¥69。**

理由：
- 语言学习工具的心理价位锚定在"一本书的钱"（¥40-60），一次性买断比订阅好卖太多，用户不用担心忘记取消
- 你的边际成本≈0，卖多卖少都不加钱，所以定低价冲量比定高价好
- 对比：市面同类日语 App 会员多在 ¥88-198/年，你定 ¥39-59 有明显价格优势，可以直接在文案里打这个对比

**技术上怎么实现付费区分**：不需要做后端账号系统。沿用你现在的邀请码机制，把 `config.json` 里的码分成 `freeCodes` / `proCodes` 两个数组，前端按码的类型决定解锁哪些模块。收款用微信/支付宝个人收款码，人工发码（1000 粉阶段完全扛得住）。

### 阶段三：5000 粉以后，产品化

- 上小红书店铺（专业号 → 虚拟商品），自动发货，不用手动发码
- 加卖：定制词表、1v1 陪练、社群年费
- 这时候再考虑做真正的账号系统

### ⚠️ 一个必须提醒的合规点

个人主体、无 ICP 经营性备案的网站**直接收费属于灰色地带**。到阶段二时更稳妥的做法是：
- 不说"买网站"，说"**赞助/请我喝咖啡，赠送 Pro 邀请码**"
- 或走小红书官方店铺（有平台背书，走正规交易）

## 三、涨粉打法：内容三分法

你现在的问题是"发网站宣传视频没人看"——这是必然的。**宣传视频永远不会是爆款。**

正确的结构是 **7 : 2 : 1**：

| 类型 | 占比 | 作用 | 你的现有资源 |
|---|---|---|---|
| **① 流量内容** | 70% | 纯干货/情绪，只为爆，**完全不提网站** | 场景对话、专业名词科普（你那条 6000 播放的日常对话就是这类） |
| **② 信任内容** | 20% | 建立"她是真懂日语的人" | 学习方法、踩坑、你自己的学习记录 |
| **③ 转化内容** | 10% | 才提网站，且**只在有流量内容托底时发** | 网站演示、用户反馈截图 |

**每 10 条视频里，最多 1 条是纯宣传。** 而且宣传视频要发在你刚出爆款之后的 1-2 天内（那时账号权重和新粉最多）。

### 你已验证的两个信号

- **日常对话 6000 播放** → 场景对话是你的爆款母题，**加倍投入**
- **科普 1000 播放** → 专业名词科普是及格线内容，当稳定输出用，别当主力

所以：**每周 5 条场景对话 + 1 条科普 + 1 条转化/信任**。

### 爆款视频的四个硬指标

根据平台规律，视频笔记能不能起量看四个数：

1. **封面点击率** —— 占成败 50%。大字号（60pt+）、高对比、**一眼看懂是"学什么"**
2. **前 3 秒留存** —— 开头必须是钩子，不是"大家好我是XX"
3. **完播 30 秒** —— 2026 年平台重点扶持中长视频，**超过 30 秒完播就能进更大流量池**。所以视频做 45-70 秒最划算，不要做 15 秒
4. **收藏/点赞比** —— 语言学习类天然高收藏，这是你的优势赛道。刻意设计"值得收藏"的信息密度（一条视频给 6-8 个词，不要只给 2 个）

### 标题公式（直接套）

| 类型 | 模板 | 示例 |
|---|---|---|
| 数字型 | 「N个XX，第M个我到日本才知道」 | 8个打羽毛球的日语，第5个中国人全说错 |
| 反认知型 | 「XX千万别说成YY」 | 在日本点餐千万别说「これください」 |
| 场景型 | 「在日本XX时，日本人其实这么说」 | 在日本理发时，日本人其实这么说 |
| 圈人群型 | 「N级党/留学生/去日本旅游的人必看」 | 去日本旅游只背这9句就够了 |
| 不看会亏型 | 「学了3年日语才发现的XX」 | 学了3年日语才发现，「すみません」有4种用法 |

### 系列化（涨粉的真正机关）

单条爆款只带来播放，**只有"系列"才带来关注**。
每条视频标题里带 **「XX系列 第N期」**，让用户觉得"关注她能持续拿到"。

建议开三个系列：
- **『在日本这样说』** —— 场景对话（主力，你的爆款母题）
- **『运动日语』** —— 专业名词（已有 sports-vocab-video skill 可以直接产）
- **『一个词的N种说法』** —— 单词深挖（高收藏）

## 四、怎么挂链接（合规路径）

小红书对外链管控很严，**正文里放 URL 会直接限流**。可用路径按安全度排：

| 路径 | 做法 | 安全度 | 转化率 |
|---|---|---|---|
| **① 私信关键词**（主力） | 视频结尾 + 评论区置顶引导「私信『日语』领取」，用私信通设自动回复 | 高 | 高 |
| **② 粉丝群** | 建群，群里发链接。官方对群管控最松 | 高 | 高 |
| **③ 个人简介** | 简介写「主页置顶有免费工具」，不放完整 URL | 高 | 中 |
| **④ 置顶笔记** | 一条图文笔记做产品介绍，图片里放二维码 | 中 | 中 |
| **⑤ 评论区** | 自己小号在评论区问"链接在哪"，主号回复引导 | 中 | 中 |
| **⑥ 小号引流** | 简介 @ 一个小号，小号做承接 | 中 | 中 |
| ⑦ 正文直接放 URL | —— | **低，别做** | —— |

**推荐组合：①+②+④**

具体话术（视频最后 3 秒 + 评论区置顶）：
> 「这个词表我整理进了一个免费的日语跟读网站，**关注后私信『日语』**，发你邀请码 ✌️」

注意：**要求"关注后私信"而不是只私信**，这一步把播放量直接变成粉丝。

## 五、别人是怎么做成的（可抄的做法）

从公开案例里能提炼出的共性：

1. **有独立开发者做 AI 学日语网站，靠小红书跑通了「种草 → 转化 → 用户反馈 → 迭代」闭环，收入过万。**核心不是网站多强，是**把用户反馈当内容素材**——用户提的需求，改完了再发一条"有人说想要XX，我加上了"，这类视频既是宣传又是干货，还显得产品在活着。**这一招你现在就能用，成本极低。**
2. **英语阅读类账号**用「AI 生成文案 + AI 配图 + Canva 排版」的极低成本流水线，从 0 做到 10 万粉——说明**画面不需要高级，需要的是稳定日更 + 统一封面模板**。
3. **虚拟资料类卖家**的通用打法：低价（9.9-39）+ 封面统一配色 + 大量铺量 + 私信自动发货。铺量比精雕重要。
4. **日语赛道优质账号稀缺**（相比英语内容饱和），场景化内容的完播率比讲课式视频高约 3 倍 —— 这是你选对赛道的部分，别放弃场景对话这个方向。

## 六、30 天执行表

### 第 1 周：修地基（先别急着日更）
- [ ] 统一封面模板（固定配色 + 固定字体位置 + 左上角固定系列名），以后所有视频套用
- [ ] 改简介：「日语学习工具开发者 | 每天一个日本人真实说法 | 主页置顶有免费工具」
- [ ] 建微信粉丝群，做好私信自动回复话术
- [ ] 把音频迁到腾讯云 COS（降备案风险）
- [ ] 盯 ICP 备案进度

### 第 2-3 周：只涨粉，不推产品
- [ ] 日更 1 条，全部是 ①流量内容（场景对话为主）
- [ ] 每条都标『系列 第N期』
- [ ] 每条视频发布后 30 分钟内自己在评论区置顶一条引导语
- [ ] 记录每条的：播放/点赞/收藏/涨粉，找出爆的那条的共同点

### 第 4 周：备案通了就开闸
- [ ] 挑最近数据最好的一条爆款，2 天后发第一条转化视频
- [ ] 转化视频结构：**先给干货（前 40 秒和普通视频一样）→ 最后 10 秒才说"这些我整理进了一个免费网站"**
- [ ] 私信发码，同时拉群
- [ ] 收集前 20 个用户的反馈截图，攒素材

### 复盘指标（每周看一次）
- 涨粉数 / 播放量（**这个比值比播放量本身重要**）
- 收藏 ÷ 点赞（>0.8 说明干货够硬）
- 私信领码人数 ÷ 播放量（转化漏斗）
